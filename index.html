Conóceme

Soy un joven de 25 años apasionado del desarrollo web, tanto Front-End como Back-End. Utilizo Responsive-Design y preprocesador CSS LESS. Desarrollo aplicaciones móviles y web con HTML5/CSS3, Phonegap, Javascript (jQuery, Backbone). En el lado del servidor utilizo REST API y PHP con frameworks como Yii o Slim, siempre procurando trabajar con patrón MVC. Persisto los datos con MYSQL. Tambien tengo experiencia con aplicaciones de escritorio en entorno Java. Cualquier proyecto que abarco está analizado previamente mediante UML. Desarrollo videojuegos 2D con el engine Unity 3D y lenguaje C#, aunque tambien he desarrollado juegos con Corona SDK (LUA) y GMStudio(~Deplhi).

Tengo otro perfil más artistico, que me permite realizar todo el acabado gráfico de un videojuego 2D, ilustraciones para publicaciones, Character Design y Concept-Art para videojuegos, animaciones 2D para cortos publicitarios y videojuegos, diseño de UI para aplicaciones web, Storyboards, diseño de camisetas, asi como Branding Corporativo para empresas, logotipos y cartelería. Trabajo con la Suite de Adobe, tableta digitalizadora Wacom y mesa de luz, procurando trabajar con el mayor mimo y dedicación posible.



Una vez mi padre me dijo: "Considera tu trabajo como un hobby. Haz lo que te apasione".
El día que deje de disfrutar con lo que hago, me haré jardinero.

"Dios está en los detalles." - Clean Code

El juego de género Casual insiste en la mecánica tap-tap, con el fin de eliminar a los diferentes insectos que se acercan a la flor central. El desarrollo se alargó a lo largo de 6 meses, y fué cancelado por parte de la agencia digital, a 2 meses de ser finalizado.
Todo el proyecto fué realizado por mí, desde el diseño de personajes, la elaboración de animaciones y sprites, así como la codificación en LUA, entre otros apartados.

  • Cliente: Seotec Agencia Digital
  • Fecha:Abril 2014
  • Github: Insect-Invasion
